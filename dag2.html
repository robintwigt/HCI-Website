We zijn de dag begonnen met een korte intro van de docent. Na dit praatje kon de daily standup van start gaan. Iedereen vertelde in het kort wat diegene gister gedaan had. Vervolgens gingen we bedenken wat er vandaag allemaal gedaan kon worden. Deze taken schreven we op post-its en plakten we onder het kopje 'to do'.

Daarna zijn we gestart met het testen van verschillende sensoren met Arduino. We hebben een aantal van de sensoren verdeeld over onze groepsleden. De sensoren die we wilden testen waren de: afstandssensor, geluidssensor, temperatuursensor, versnellingsmeter en lichtsensor. Ik ben gaan werken met de temperatuursensor. Er waren 3 verschillende temperatuursensoren: ‘Temp’, ‘Analog temp’ en ‘Digital temp’.

Bij alle 3 de sensoren kreeg ik wel waarden te zien, alleen was het niet de waarde die ik wilde. De waarde die ik wilde was namelijk graden celsius. Ik kon er niet achter komen welke waarden de sensoren gaven. Ik dacht eerst aan Fahrenheit, alleen toen ik dit omrekende in Arduino kwamen er getallen uit die ook niet helemaal leken te kloppen. Het zou dan namelijk 26 graden zijn en dat was het niet op dat moment. Dit kan echter ook liggen aan de nauwkeurigheid van de sensor. 

De code die ik gebruikt het voor de temperatuursensor is hieronder te zien. De 3 regels die ik in commentaar heb gezet, waren probeersels van berekeningen om de juiste waarde eruit te krijgen. De waarden die daaruit kwamen waren dus niet helemaal realistisch.

Na de presentatie zijn we met de hele groep nog even gaan brainstormen over het concept. We hebben besloten om het een beetje aan te passen. Bij het concept is het nu de bedoeling dat 2 mensen met elkaar gaan samenwerken door middel van een spel. Persoon 1 gaat lopen door een doolhof met een blinddoek om en 2 sensoren, 1 sensor op de linkerhand en 1 sensor op de rechterhand. Als de tweede persoon op een button klikt met een linkerpijl, dan gaat de sensor op de linkerhand trillen. Als de persoon op een button klikt met een rechterpijl, dan gaat de sensor op de rechterhand trillen. Hiermee kan persoon 2 persoon 1 begeleiden door het doolhof.

Het concept hadden we helemaal uitgedacht. Daarom heb ik een snelle schets gemaakt van het concept. Hierdoor is het concept duidelijk gevisualiseerd en zullen er geen onduidelijkheden zijn over het concept .

Voor ons concept hebben we een sensor nodig die kan trillen. Deze sensor was echter niet aanwezig, dus moesten we nog bestellen. We konden vandaag dus nog niet experimenteren met deze sensor. De code die we nodig gaan hebben kan wel alvast geschreven worden, alleen dan met een andere sensor in plaats van de sensor die kan trillen.

Ik heb een code geschreven in Arduino die werkt met 2 buttons en 2 LEDs. Als je de linkerbutton indrukt gaat de linker LED aan en als je de rechterbutton indrukt gaat de rechter LED aan. Deze code kunnen we later gaan toepassen bij de tril sensor. Dat gaat namelijk op dezelfde manier werken: als de linkerbutton ingedrukt wordt, gaat de linker sensor trillen en andersom. De code die ik heb geschreven is hieronder te lezen en het resultaat daarvan is daaronder te zien.

// constants won't change. They're used here to set pin numbers:
const int buttonPin1 = 2; // the number of the pushbutton pin
const int buttonPin2 = 3; // the number of the pushbutton pin
const int ledPin1 = 13; // the number of the LED pin
const int ledPin2 = 12; // the number of the LED pin

void setup() {
// initialize the LED pin as an output:
pinMode(ledPin1, OUTPUT);
// initialize the LED pin as an output:
pinMode(ledPin2, OUTPUT);
// initialize the pushbutton pin as an input:
pinMode(buttonPin1, INPUT);
// initialize the pushbutton pin as an input:
pinMode(buttonPin2, INPUT);
}

void loop() {
// check if the pushbutton is pressed. If it is, the buttonState is HIGH:
if ((digitalRead(buttonPin1) == HIGH) && (digitalRead(buttonPin2) == HIGH)) {
// turn LED on:
digitalWrite(ledPin1, HIGH);
digitalWrite(ledPin2, HIGH);
}
else if (digitalRead(buttonPin1) == HIGH) {
// turn LED on:
digitalWrite(ledPin1, HIGH);
digitalWrite(ledPin2, LOW);
}
else if (digitalRead(buttonPin2) == HIGH) {
// turn LED on:
digitalWrite(ledPin1, LOW);
digitalWrite(ledPin2, HIGH);
}
else {
// turn LED off:
digitalWrite(ledPin1, LOW);
digitalWrite(ledPin2, LOW);
}
}
