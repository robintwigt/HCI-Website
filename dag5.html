Dag 5 - vrijdag 7 juni

De dag begon met een presentatie van Marcello Gómez Maureira. Zijn presentatie ging over Designing Playful Interactions.

Na de presentatie heeft Batuhan de requirements en essentials list opgesteld. De requirements zijn:

Co-operative gameplay
Wireless connection between the Lilypad and the Vibe Boards
When pressing the buttons on the controller the bracelet vibrates in the corresponding direction
Arcade theme
A controller for the guide to be able to direct the guided
Bracelets with the Lilypad Vibe Boards
A maze
Start and Finish board hanging from the ceiling
A working stopwatch to keep track of time
A leaderboard for times
The guided has to be blindfolded and deafened

De essentials zijn:

Wooden boards
Paint
Buttons
Lilypad
Arduino
Lilypad Vibe Board
Leather for the bracelets
LCD Screen
Whiteboard with markers
Either tape or wood for the maze
Wireless adapters for the Vibe Board and Lilypad
Wemos Wifi Arduino
Wireless headphones
Arcade music playlist
Blindfolds
Boards to hang from the ceiling

Ondertussen ben ik aan de slag gegaan met de timer. Gisteren (dag 4) hadden we namelijk bedacht dat het erg leuk zou zijn als de speler zelf de tijd bij kan houden in de arcade box. Anders zouden wij dat zelf moeten doen door middel van de stopwatch op onze mobiel, maar dat is veel minder leuk. Uiteindelijk is het gelukt om de timer werkend te krijgen. De code die ik hiervoor heb gebruikt is hieronder te lezen.

#include < LiquidCrystal.h>
LiquidCrystal lcd(7, 6, 5, 4, 3, 2);

void setup() {
lcd.begin(16, 2);
lcd.clear();

Serial.begin(9600);

pinMode(8, INPUT);
digitalWrite(8, HIGH);
pinMode(9, INPUT);
digitalWrite(9, HIGH);
}

double i = 0;
double a = millis();
double c ;

void loop() {
lcd.clear();
lcd.print("press start");
delay(100);

if(digitalRead(8) == LOW) {
lcd.clear();
a = millis();
while(digitalRead(9) == HIGH) {
c = millis();
i = (c - a) / 1000;
lcd.print(i);
lcd.setCursor(7,0);
lcd.print("Seconds");
lcd.setCursor(0,0);
Serial.println(c);
Serial.println(a);
Serial.println(i);
Serial.println("......");
delay(100);
}

if(digitalRead(9) == LOW) {
while(digitalRead(8) == HIGH) {
lcd.setCursor(0,0);
lcd.print(i);
lcd.setCursor(11,0);
lcd.print("");
lcd.setCursor(0,0);
delay(100);
}
}
}
}

Het scherm zat nu op het Arduino bord en dat is niet handig als we het scherm later in de Arcade box willen verwerken. Ik heb het scherm er dus af gehaald en verbonden aan het bord door gebruik te maken van male-to-female kabels. Hieronder een foto van hoe het scherm is verbonden met het Arduino bord door middel van de male-to-female kabels en een video van dat het met deze kabels nog steeds werkt.

Vervolgens hebben we opgesteld wat er nog allemaal gekocht moest worden en wat er gedaan moest worden. Hierdoor kregen we een helder overzicht en wist iedereen waar we aan toe waren. Een van mijn groepsleden heeft ook al meteen een aantal dingen gekocht, bijvoorbeeld het hout voor de Arcade box. Ons idee is nu namelijk om een Arcade box te maken die om de nek hangt van persoon 2. Het gaat namelijk niet lukken om het draadloos te maken, dus moet de besturing draagbaar worden. We vonden de Arcade machine zo leuk eruit zien, dat we ervoor hebben gekozen om deze toch te maken, maar dan een versie die je om je nek kunt hangen.

Daarnaast hebben we gedacht over een naam van ons concept. We willen deze naam ook op de Arcade box zetten, dus moet het een leuke naam zijn. De naam ‘You and Eye’ is het uiteindelijk geworden, omdat je het spel met zijn tweeën speelt en omdat de ene persoon het oog is van de andere persoon. Dit vond iedereen de geschikste en leukste naam voor ons project.

De Arcade box willen we zwart gaan verven en we willen er stickers op plakken. Ik ben daarom begonnen met het zoeken naar leuke stickers die we op de box kunnen plakken. Alle stickers heb ik in een Word document gezet, zodat we het later kunnen uitprinten. Hieronder is een afbeelding te zien van de stickers.

We hebben daarna de dag doorgenomen bij het scrum bord.
